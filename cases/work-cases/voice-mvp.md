# MVP сервиса для голосового общения

Инициировал проверку идеи через максимально простой интерфейс и одну ключевую функцию — голосовую связь.

## Что было сделано

- Сформулировал концепцию «дешевого» теста гипотезы без перегруженной функциональности.
- Предложил одноразовые сессии общения без хранения лишних данных.
- Получили ранний фидбек аудитории и провели глубинные интервью.
- На основе данных собрали первые UX-сценарии и интерфейсные решения.
- Подготовил интерактивные прототипы для тестирования с фокус-группой.

## Итог

Быстро подтвердили ценность идеи и получили базу для следующего продуктового этапа без дорогой реализации на старте.
