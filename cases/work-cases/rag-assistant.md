# AI-ассистент для работы с документами

Проект по созданию MVP сервиса, который помогает работать с финансовыми документами с помощью RAG-подхода.

## Формат работы

Проектный спринт, в котором быстро собрали основной сценарий взаимодействия и интерфейсы ключевых состояний.

## Что сделал

- Участвовал в анализе задачи и формировании продуктовых гипотез.
- Проработал основные сценарии и пользовательский флоу.
- Спроектировал и отрисовал важные модалки, состояния и переходы.
- Подготовил спецификацию и комментарии для передачи в разработку.

## Результат

Команда получила готовую визуальную и логическую основу MVP, которую можно было сразу брать в реализацию и тест.
