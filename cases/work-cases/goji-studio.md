# goji.studio

Сейчас проект на активной стадии развития: обновляем позиционирование, усиливаем сценарии взаимодействия и запускаем дополнительные продуктовые страницы.

![goji.studio — редизайн сайта](goji-site.png)

## Роль

Senior web designer на проекте.

## Что было сделано

- Помогал с ресерчем, конкурентным анализом и формированием гипотез.
- Работали итерациями: собирали структуру, дорабатывали логику экранов и блоков.
- Сделал несколько дизайн-концепций, затем собрал целостный визуальный язык.
- Подготовил макеты для разработки и сопровождал верстку.
- Разделили мобильные и десктопные сценарии, чтобы точнее управлять UX.
- По метрикам (сессии, скролл, хитмапы) запустили следующий цикл редизайна.
